


# 需要配置的端口号(按需)

| 组件               | 默认端口号                                            |
| ------------------ | -------------------------------------------------- |
| API Server         | 6443                                               |
| Controller Manager | 10257                                              |
| Scheduler          | 10259                                              |
| kubelet            | 10250、10255(只读端口号)                             |
| etcd               | 2379(供客户端访问)、2380(供etcd集群内节点之间访问)       |
| 集群的 DNS 服务      | 53(TCP/UDP)                                        |
| calico             | 179                                                |
